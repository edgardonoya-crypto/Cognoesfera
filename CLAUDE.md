# Cognoesfera — Sistema Vivo del Paradigma Aleph

## Qué es este proyecto
Aplicación que materializa el Paradigma Aleph — un marco para la emergencia de Inteligencia Colectiva. La tecnología sigue al paradigma, no al revés.

## Stack
- Next.js + TypeScript
- Supabase (base de datos + auth)
- Vercel (deploy automático)
- URL: cognoesfera.vercel.app

## Principio rector
Cada elemento crea condiciones para que algo se vea — no explica lo que debería verse.

## Próxima tarea
Migrar el prototipo HTML estático a Next.js conectado a Supabase, empezando por login real con autenticación.
